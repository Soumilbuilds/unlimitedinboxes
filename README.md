# Unlimited Mailboxes Final

Local dev instructions:
1. Install server deps: `cd server && npm install`
2. Install client deps: `cd ../client && npm install`
3. Start server: `cd ../server && npm run dev`
4. Start client: `cd ../client && npm run dev`
5. Open http://localhost:5173

Default demo login:
- admin@unlimitedinboxes.com / admin account

.env is already copied into `server/.env`.
